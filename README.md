# Kids Adventure Game

Welkom bij de Kids Adventure Game! Dit is een verzameling interactieve spellen ontworpen voor kinderen om hun vaardigheden te verbeteren.

## Wat is Kids Adventure Game?
Het is een educatief platform dat drie verschillende games biedt: **Rekenen**, **Woorden** en **Memory**. Elk spel is gebouwd met pure HTML, CSS en JavaScript om een snelle en educatieve ervaring te bieden.

## Doel van het Game
Het hoofddoel van dit project is om leren leuker te maken door middel van interactieve uitdagingen. Of het nu gaat om het oefenen van sommen, het verbeteren van de woordenschat of het trainen van het geheugen, dit platform helpt kinderen spelenderwijs te leren.

## Hoe kan je spelen?
1. Open het bestand `index.html` in je favoriete webbrowser.
2. Je komt op het startscherm waar je de verschillende spellen kunt zien.
3. Klik op de knop **"Speel Nu"** bij het spel dat je wilt doen.
4. Volg de instructies in elk spel om te winnen!

## Hoeveel gamen zijn er?
Er zijn in totaal **3** games aanwezig in dit project:
1.  **Rekenen**: Los wiskundige sommetjes op en verbeter je rekenvaardigheid.
2.  **Woorden Game**: Ontdek en oefen met verschillende woorden om je taalgebruik te versterken.
3.  **Memory Game**: Test je hersenen en daag je geheugen uit door de juiste paren bij elkaar te zoeken.
