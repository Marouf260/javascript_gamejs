# 🎮 Kids Adventure Game — Deluxe Edition

Welkom bij de **Kids Adventure Game**, een luxe en interactief educatief platform ontworpen om leren leuk en uitdagend te maken voor kinderen. Dit project is ontwikkeld als een professionele showcase van moderne webvaardigheden (HTML5, CSS3, JS).

---

## ✨ Over het Project
Dit platform biedt een **Premium Gaming Experience** waarbij educatie centraal staat. De interface is geïnspireerd op moderne gaming-portals, compleet met glas-morfisme, dynamische gradiënten en vloeiende animaties.

### 🧩 Beschikbare Games
Momenteel bevat het platform **3 unieke werelden**:

1.  **🧮 Rekenen (Math Quest)**: 
    *   Train je hersenen met sommen (+, -, ×, ÷).
    *   Meerdere niveaus: van basisonderwijs tot expert.
    *   Inclusief timer en levens voor een echte arcade-ervaring!

2.  **✍️ Woorden (Word Scramble)**:
    *   Verbeter je woordenschat door gehusselde woorden op te lossen.
    *   Thema's zoals: Dieren, Vervoer, Wonen en Familie.
    *   Inclusief hints en een progressie-systeem.

3.  **🧠 Memory (Mind Match)**:
    *   Test je visuele geheugen door paren te matchen.
    *   Prachtige iconen en soepele kaart-animaties.
    *   Verschillende thema's en moeilijkheidsgraden.

---

## 🚀 Belangrijkste Features
*   **Premium Design**: Een 'Dark Mode' gaming look met neon accenten en Google Fonts (Orbitron & Montserrat).
*   **DRY Codebase**: Geoptimaliseerde CSS en JavaScript voor maximale snelheid en onderhoudbaarheid.
*   **Responsief**: Geoptimaliseerd voor zowel desktop als mobiele apparaten.
*   **Audio-Feedback**: Interactieve geluidseffecten bij elke actie (clicks, winst, tijd).
*   **Hoogste Score**: Wordt lokaal opgeslagen (LocalStorage) zodat je altijd je eigen record kunt verbeteren.

---

## 🛠️ Tech Stack
*   **Structuur**: HTML5 (Semantisch)
*   **Styling**: Vanilla CSS3 (Custom Properties, Flexbox, Grid, Animaties)
*   **Logica**: Modern JavaScript (ES6+)
*   **Tools**: Git, LocalStorage API, Audio API
